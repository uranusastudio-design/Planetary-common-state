# WMO

## Scientific Role

The World Meteorological Organization coordinates global meteorological, climate, hydrological, and atmospheric observation standards and services.

## Available Observation Categories

- Weather
- Climate
- Hydrology
- Atmospheric composition
- Extreme events
- Observation standards

## Update Frequency

WMO-associated data cadence varies by contributing service, from operational observations to climate summaries and standards documentation.

## Future PCS Integration

WMO is a candidate reference source for standards, climate indicators, hydrological context, and atmospheric observations. No API is implemented in this registry entry.
