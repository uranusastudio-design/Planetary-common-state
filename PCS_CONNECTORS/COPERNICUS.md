# Copernicus

## Scientific Role

Copernicus provides European Earth-observation services for atmosphere, climate, land, marine, emergency, security, and environmental monitoring.

## Available Observation Categories

- Atmosphere
- Climate reanalysis
- Land cover
- Marine state
- Cryosphere
- Vegetation
- Hydrology

## Update Frequency

Copernicus products vary by service and include near-real-time, daily, monthly, seasonal, and reanalysis products.

## Future PCS Integration

Copernicus is a candidate provider for multi-domain PCS observations and validation datasets. No API is implemented in this registry entry.
