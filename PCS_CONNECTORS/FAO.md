# FAO

## Scientific Role

FAO provides global food, agriculture, forestry, fisheries, land, and food-security information.

## Available Observation Categories

- Agriculture
- Crop
- Livestock
- Fisheries
- Forestry
- Food supply
- Food security
- Land use

## Update Frequency

FAO products vary from annual statistical releases to periodic monitoring and assessment products.

## Future PCS Integration

FAO is a candidate provider for food-system, biosphere, land-use, and human-system observations. No API is implemented in this registry entry.
