# GBIF

## Scientific Role

GBIF provides global biodiversity occurrence data and supports access to species-observation records from distributed institutions.

## Available Observation Categories

- Species occurrence
- Biodiversity
- Taxonomic records
- Ecological observations
- Spatial biodiversity data

## Update Frequency

GBIF updates depend on contributing institutions and data publishers, with ongoing additions and revisions.

## Future PCS Integration

GBIF is a candidate provider for biodiversity and biosphere-related PCS observations. No API is implemented in this registry entry.
