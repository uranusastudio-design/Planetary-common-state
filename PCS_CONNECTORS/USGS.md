# USGS

## Scientific Role

USGS provides land, water, geology, hazards, remote sensing, and natural-resource observations relevant to PCS.

## Available Observation Categories

- Land cover
- Hydrology
- Streamflow
- Geology
- Earthquakes
- Volcanoes
- Terrain
- Remote sensing archives

## Update Frequency

USGS data cadence ranges from real-time hazard and stream observations to periodic surveys and long-term archives.

## Future PCS Integration

USGS is a candidate provider for hydrology, geosphere, land-cover, hazard, and surface-change observations. No API is implemented in this registry entry.
