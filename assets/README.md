# PCS Asset Library

This directory reserves the future asset structure for PCS Observatory visual, audio, animation, imagery, and model resources.

This milestone creates folders and documentation only. No audio, imagery, animation, model, scientific data, or generated media files are included.

Large future assets should be stored in Cloudflare R2 or another dedicated object-storage service, not committed to GitHub.

## Categories

- `audio/`: future alert, voice, ambient, and interface sounds.
- `animations/`: future UI and data-state animation assets.
- `imagery/`: future terrain, satellite, cloud, and overlay imagery.
- `models/`: future Earth, Moon, Mars, satellite, and solar-system models.
