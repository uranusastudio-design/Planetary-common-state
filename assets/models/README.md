# PCS Model Assets

This directory reserves future 3D model assets for PCS Observatory.

No 3D models are included in this milestone. The current Observatory continues to use the existing Cesium Earth visualization only.

Large future model assets should be stored in Cloudflare R2, not GitHub.

## Reserved Subfolders

- `earth/`
- `moon/`
- `mars/`
- `satellites/`
- `solar_system/`
