# Library Schema

Each method entry should use the following fields:

- Method ID
- Method Name
- Method Family
- Scientific Area
- Applicable PCS Module
- Purpose
- Inputs
- Outputs
- Assumptions
- Limitations
- Validation Requirement
- Implementation Status
- References
- Notes

## Use

The schema keeps method documentation consistent across mathematics, physics, Earth-system science, computational science, validation, AI-assisted review, and visualization.
