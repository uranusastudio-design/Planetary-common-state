# PCS Scientific Methods Library v1.0

The PCS Scientific Methods Library is the shared methods layer for the Planetary Common State platform.

It supports:

- computation;
- verification;
- validation;
- model checking;
- uncertainty analysis;
- residual analysis;
- data assimilation;
- visualization;
- AI-assisted scientific review;
- research reproducibility.

## Module Boundary

This library is separate from PCS_ENGINE, PCS_CONNECTORS, PCS_OBSERVATORY, and PCS_VARIABLE_REGISTRY.

- PCS_ENGINE computes.
- PCS_CONNECTORS import data.
- PCS_OBSERVATORY displays.
- PCS_SCIENTIFIC_LIBRARY documents methods.

## Scientific Boundary

This library is not a problem list. It does not claim to solve Hilbert Problems, Millennium Problems, or any open mathematical problem. It maps mathematical, physical, computational, Earth-system, AI, visualization, and validation methods into PCS research tool families.

## Implementation Boundary

Documentation only. No code, calculations, APIs, fake data, or scientific output are implemented here.
