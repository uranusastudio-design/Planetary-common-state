# Space Environment Datasets

| Observation Category | Potential Dataset | Potential Providers | Expected Update Frequency | Potential Resolution | Future PCS Variables |
|---|---|---|---|---|---|
| Solar Activity | GOES X-ray flux, SDO observations | NOAA SWPC, NASA | Real-time to daily | Time series/imagery | Solar activity |
| Solar Wind | DSCOVR, ACE, OMNIWeb | NOAA SWPC, NASA | Real-time to hourly/monthly | Time series | Solar wind context |
| Geomagnetic Activity | Kp index, Dst index | NOAA SWPC, GFZ, WDC Kyoto | Near-real-time to final indices | Index time series | Geomagnetic activity |
| Space Weather Alerts | NOAA SWPC alerts and watches | NOAA SWPC | Real-time/event-based | Event records | Space-weather status |
| Near Earth Objects | NASA CNEOS close-approach data | NASA CNEOS | Continuous updates | Object/event records | Near-Earth object context |
| Planetary Environment | NASA planetary data products | NASA Planetary Data System | Mission/product dependent | Mission dependent | Planetary context |
| Radiation Environment | GOES particle flux, NASA radiation models | NOAA SWPC, NASA | Real-time to periodic | Time series/model products | Radiation environment |
| Ionosphere | GNSS TEC products, COSMIC radio occultation | NASA, NOAA, UCAR, international GNSS services | Near-real-time to daily | Global/regional gridded | Ionospheric context |
