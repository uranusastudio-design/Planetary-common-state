# Space Environment

## 1. Scientific Scope

The Space Environment domain covers external space-weather and solar-terrestrial conditions that can affect planetary observations, satellites, communication systems, and infrastructure.

## 2. Major Subdomains

- Solar Activity
- Geomagnetic Activity
- Radiation Environment
- Ionosphere
- Space Weather
- Satellite Environment
- Solar Wind
- Magnetosphere

## 3. Primary Scientific Agencies

- NASA
- NOAA
- ESA
- JAXA
- WMO

## 4. Role inside PCS

Space Environment variables provide context for satellite observation quality, technological risk, communication resilience, and external forcing relevant to the Planetary Common State.
