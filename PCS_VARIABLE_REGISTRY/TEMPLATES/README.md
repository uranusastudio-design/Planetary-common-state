# PCS Variable Template Library

The PCS Variable Template Library defines the official documentation templates for future Planetary Common State variables, observations, providers, and quality-control metadata.

Every future PCS variable must follow these templates before it is integrated into the PCS Variable Registry, PCS Engine, PCS Observatory, connectors, or scientific publications.

The templates guarantee consistency across hundreds of future variables by standardizing:

- variable definitions
- observation metadata
- provider metadata
- quality-control metadata
- references and provenance
- confidence and review status

This library is documentation only. It does not implement variables, APIs, engine logic, dashboard logic, connectors, or computation.
