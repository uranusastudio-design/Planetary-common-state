# PCS Scientific Taxonomy

This document defines the scientific classification hierarchy for the PCS Variable Registry.

```text
Planetary Common State
  -> Scientific Domains
    -> Subdomains
      -> Future Variables
```

No variables are listed in this document. The categories below define the structure into which future variables will be registered.

## 1. Atmosphere

### Scientific Scope

The Atmosphere domain covers the physical, chemical, radiative, and dynamical state of Earth's atmosphere.

### Role inside PCS

Atmospheric categories provide the thermal, radiative, chemical, and circulation context needed to represent planetary-scale environmental state.

### Future Variable Categories

- Temperature
- Pressure
- Humidity
- Clouds
- Wind
- Atmospheric Dynamics
- Radiation
- Atmospheric Chemistry
- Trace Gases
- Aerosols
- Extreme Weather

### Future Observation Sources

- NASA
- NOAA
- Copernicus
- ESA
- JAXA
- WMO
