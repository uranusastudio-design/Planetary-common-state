# Food System

## 1. Scientific Scope

The Food System domain covers agricultural production, food availability, crop condition, fisheries, livestock, food security, and land-use pressures related to food systems.

## 2. Major Subdomains

- Crop Production
- Crop Condition
- Food Security
- Fisheries
- Livestock
- Agricultural Land Use
- Soil Condition
- Supply Stability

## 3. Primary Scientific Agencies

- FAO
- NASA
- USDA
- World Bank
- WMO
- Copernicus

## 4. Role inside PCS

Food System variables connect biosphere, hydrology, climate, energy, and human-system conditions to food availability, resilience, and long-term societal viability.
