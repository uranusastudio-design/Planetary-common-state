# Geosphere

## 1. Scientific Scope

The Geosphere domain covers solid-Earth and near-surface geophysical conditions, including lithosphere, topography, gravity, tectonic activity, geomorphology, and geological boundary conditions.

## 2. Major Subdomains

- Topography
- Gravity
- Tectonics
- Lithosphere
- Geomorphology
- Volcanic Activity
- Seismic Activity
- Geodesy

## 3. Primary Scientific Agencies

- NASA
- ESA
- USGS
- NOAA
- JAXA

## 4. Role inside PCS

Geosphere variables provide boundary conditions, mass redistribution context, terrain constraints, and geophysical state information for the broader Planetary Common State.
