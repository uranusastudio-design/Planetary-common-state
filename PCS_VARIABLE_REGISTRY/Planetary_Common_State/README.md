# Planetary Common State

## 1. Scientific Scope

The Planetary Common State domain covers integrated PCS variables, aggregate state descriptors, projection metadata, coverage indicators, confidence summaries, and cross-domain state products.

## 2. Major Subdomains

- PCS State
- Projection Coverage
- Confidence
- Data Quality
- Cross-domain Coupling
- Validation Status
- State History
- Observatory Outputs

## 3. Primary Scientific Agencies

- NASA
- NOAA
- Copernicus
- ESA
- WMO
- UN
- World Bank

## 4. Role inside PCS

Planetary Common State variables organize the integrated state representation used by PCS. They provide the bridge between domain observations, projection definitions, validation, and observatory outputs.
