# Human System

## 1. Scientific Scope

The Human System domain covers social, demographic, economic, health, governance, and adaptive-capacity observations relevant to coupled human-Earth system state.

## 2. Major Subdomains

- Population
- Health
- Economy
- Governance
- Adaptive Capacity
- Urbanization
- Migration
- Social Vulnerability

## 3. Primary Scientific Agencies

- UN
- World Bank
- FAO
- WHO
- OECD
- National Statistical Agencies

## 4. Role inside PCS

Human System variables contribute information about societal exposure, adaptive capacity, infrastructure demand, resource use, and long-term viability within the Planetary Common State.
