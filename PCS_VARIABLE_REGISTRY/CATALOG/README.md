# PCS Variable Catalog

The PCS Variable Catalog is the documentation bridge between the scientific taxonomy and future implementation layers of the Planetary Common State platform.

It connects:

```text
Scientific Taxonomy
  -> Variables
    -> Observation Sources
      -> PCS Engine
```

No variables are implemented yet. This catalog only defines future variable groups that may later become formal PCS registry entries.

The catalog is intentionally documentation-only. It does not define APIs, connectors, engine code, dashboard logic, or computation.
