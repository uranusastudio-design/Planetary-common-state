# Biosphere

## Scientific Scope

The Biosphere catalog covers future variable groups describing living systems, ecological structure, biodiversity, productivity, and disturbance.

## Future Variable Groups

- Vegetation
- Forest
- Grassland
- Wetland
- Biodiversity
- Productivity
- Wildfire
- Carbon Cycle
