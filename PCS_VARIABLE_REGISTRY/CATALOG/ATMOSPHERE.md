# Atmosphere

## Scientific Scope

The Atmosphere catalog covers future variable groups describing the physical, chemical, radiative, and dynamical state of Earth's atmosphere.

## Future Variable Groups

- Temperature
- Pressure
- Humidity
- Wind
- Clouds
- Radiation
- Chemistry
- Trace Gases
- Aerosols
- Extreme Weather

## Possible Observation Sources
