# Biosphere

## 1. Scientific Scope

The Biosphere domain covers living systems and ecological structure, including vegetation, biodiversity, ecosystem productivity, habitat condition, and ecological complexity.

## 2. Major Subdomains

- Vegetation
- Biodiversity
- Ecosystem Productivity
- Habitat Condition
- Land Cover
- Species Occurrence
- Ecosystem Health
- Carbon Uptake

## 3. Primary Scientific Agencies

- NASA
- ESA
- Copernicus
- GBIF
- FAO
- USGS

## 4. Role inside PCS

Biosphere variables contribute ecological state, adaptive capacity, carbon-cycle response, and biological integrity to the Planetary Common State.
