# Cryosphere

## 1. Scientific Scope

The Cryosphere domain covers frozen components of the Earth system, including glaciers, ice sheets, sea ice, snow cover, permafrost, and frozen soil.

## 2. Major Subdomains

- Sea Ice
- Ice Sheets
- Glaciers
- Snow Cover
- Permafrost
- Ice Mass Balance
- Surface Melt
- Frozen Soil

## 3. Primary Scientific Agencies

- NASA
- NOAA
- Copernicus
- ESA
- USGS
- WMO

## 4. Role inside PCS

Cryosphere variables represent frozen-water storage, albedo feedbacks, sea-level contributions, and high-latitude state changes that influence planetary stability and long-term persistence.
